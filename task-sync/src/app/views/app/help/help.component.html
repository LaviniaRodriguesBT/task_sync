Ajuda

Como utilizar o sistema Task Sync

1. Funcionalidades Principais

1.1. Gerenciar Eventos

  • Criar Eventos: Permite ao administrador criar novos eventos, definindo nome, área de atuação, e outros detalhes importantes.
  • Editar Eventos: Atualize informações de eventos, como datas e localizações.
  • Excluir Eventos: Remova eventos duplicados ou criados erroneamente.
  • Emitir Relatórios: Gere relatórios em PDF sobre usuários e atividades.

1.2. Gerenciar Pessoas

  • Cadastrar Pessoas: Adicione novos colaboradores, incluindo dados pessoais e informações de login.
  • Editar Pessoas: Atualize informações de colaboradores já cadastrados.
  • Excluir Pessoas: Remova colaboradores que não fazem mais parte do evento.

1.3. Gerenciar Atividades

  • Criar Atividades: Adicione atividades específicas vinculadas a um evento.
  • Editar Atividades: Altere os detalhes das atividades existentes.
  • Excluir Atividades: Remova atividades criadas por engano.
  • Atribuir Atividades: Designe atividades específicas a colaboradores.

1.4. Comunicação Interna

  • Chat: Permite a comunicação entre administradores e colaboradores diretamente no sistema, com notificações para novas mensagens.

1.5. Acompanhar Andamento

  • Dashboard: Visualize o progresso das atividades de forma gráfica, com informações sobre atividades pendentes, em andamento, e concluídas.

1.6. Gerenciar Cronograma

  • Criar Cronograma: Configure o cronograma de um evento, incluindo datas, horários e responsáveis.
  • Editar Cronograma: Atualize informações de cronogramas existentes.
  • Excluir Cronograma: Remova cronogramas criados de forma incorreta.

2. Requisitos do Sistema

2.1. Requisitos Funcionais: Para detalhes específicos de cada funcionalidade, consulte a seção de Requisitos Funcionais do documento. As principais funcionalidades incluem a criação, edição, e exclusão de eventos, pessoas, atividades, e cronogramas. Clique aqui para acessar a documentação.

2.2. Requisitos Não-Funcionais: O sistema deve ser intuitivo, eficiente, confiável, e compatível com o sistema operacional Windows. Além disso, a implementação deve ser feita em Java e Angular, seguindo padrões organizacionais específicos. Clique aqui para acessar a documentação.

3. Solução de Problemas Comuns

  • Problemas de Login: Verifique se suas credenciais estão corretas e se a conta está ativa.
  • Erro ao Criar/Editar Eventos: Certifique-se de que todos os campos obrigatórios estão preenchidos corretamente.
  • Não é possível excluir atividade/pessoa: Verifique se a atividade ou pessoa está vinculada a um evento. Desvincule antes de tentar excluir.
  • Relatórios não são gerados: Confirme se você tem permissões de administrador.

4. Atualizações e Manutenção

Histórico de Versões: As atualizações do sistema estão documentadas junto ao projeto no GitHub. Clique ao repositório.

5. Termos de Uso e Privacidade

O sistema segue a Lei Geral de Proteção de Dados (LGPD), garantindo a segurança e privacidade dos dados dos usuários.
